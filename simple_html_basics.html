HTML5 Semantic Elements

This page demonstrates various HTML5 semantic elements:

  • <header> - Used for page/section headers
  • <nav> - Navigation links container
  • <main> - Main content area
  • <section> - Thematic grouping of content
  • <article> - Self-contained content
  • <aside> - Sidebar or tangential content
  • <footer> - Page/section footer
  • <figure> - Media with caption
  • <figcaption> - Caption for figure

HTML Entities & Special Characters

HTML entities are used to display special characters:

  • &lt; produces < (less than)
  • &gt; produces > (greater than)
  • &amp; produces & (ampersand)
  • &quot; produces " (quotation mark)
  • &apos; produces ' (apostrophe)
  • &nbsp; produces a non-breaking space
  • &copy; produces © (copyright)
  • &reg; produces ® (registered trademark)
  • &trade; produces ™ (trademark)
  • &euro; produces € (euro symbol)
  • &pound; produces £ (pound symbol)
  • &yen; produces ¥ (yen symbol)

Generic Containers: Div & Span

This paragraph is inside a <div> element. Divs are block-level elements used for grouping and styling. They take up the full width available and start on a new line.

You can have highlighted text using span which is an inline element that doesn't break the flow of text.

This document includes various meta elements in the <head> section:
